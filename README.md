# Test task
## Social Network
### 1.Install requirements:
```
pip install -r requirements.txt
```
### 2.Set env variables:
```
DJANGO_SECRET_KEY
```
### 3.Apply migrations:
```
python manage.py migrate
```
### 4.Run Social Network:
```
python manage.py runserver
```
